### Logo Component

Logo component is used to display the company's logo in various locations of the web application.

### Features

- Displayed as an SVG.
- Has accessibility features.

### Usage

To use the company's logo, simply import and use component:

```tsx
import { Logo } from './components/Logo';

function App() {
  return (
    <>
      <Logo />
    </>
  );
}

export default App;
```
